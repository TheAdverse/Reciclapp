# ♻️ Reciclapp
**Una solución tecnológica para la gestión de residuos en Hidalgo.**

## 🚀 Descripción
Reciclapp es una aplicación móvil desarrollada en **Flutter** y **Supabase** que permite a los usuarios trackear su reciclaje, ganar puntos y localizar centros de acopio.

## ✨ Características
* **Autenticación Segura:** Registro e inicio de sesión con Supabase.
* **Mapa de Centros:** Localización de puntos de reciclaje cercanos.
* **Sistema de Puntos:** Gamificación para incentivar el reciclaje.
* **Escáner de Residuos:** (En desarrollo) Identificación de materiales.

## 🛠️ Instalación
1. Clona el repositorio: `git clone https://github.com/tu-usuario/Reciclapp.git`
2. Instala dependencias: `flutter pub get`
3. **Importante:** Crea un archivo `.env` en la raíz con tus llaves de Supabase.
4. Ejecuta: `flutter run`
